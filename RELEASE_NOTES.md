# Release Notes

## v0.1 - Initial Release

**Release Date:** November 2024

### Overview
This is the initial release of Metrolist-Subsonic, a fork of Metrolist that adds Subsonic server integration for streaming music from your personal music server.

### Features
- **Subsonic Integration**: Connect to and stream music from any Subsonic-compatible server (Subsonic, Navidrome, Airsonic, etc.)
- **Local Playback**: All the features of the original Metrolist for local music playback
- **Multiple Architecture Support**: Universal, ARM64, ARMeabi, x86, and x86_64 builds available
- **Material Design 3**: Modern UI with Material You dynamic theming
- **Media Controls**: Full media session support with notification controls

### Technical Details
- **Min SDK**: 26 (Android 8.0)
- **Target SDK**: 36
- **Build Version Code**: 1
- **Application ID**: com.metrolist.subsonic

### Installation
Download the appropriate APK for your device architecture from the [Releases page](https://github.com/liraz747/Metrolist-Subsonic/releases/tag/v0.1).
